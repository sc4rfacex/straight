# KBMermaidSyntax

Repositorio central de knowledge para sintaxis Mermaid. Cada archivo se asocia a un hashtag de búsqueda en el prompt del sistema.

| Archivo | Hashtag sugerido | Cobertura principal |
|---|---|---|
| `flowchart.md` | `#MermaidFlowcharts` | Diagramas de flujo, subgrafos, patrones anti-error |
| `sequence.md` | `#MermaidSequence` | Interacciones y flujos de mensajería |
| `class.md` | `#MermaidClass` | UML de clases y patrones estructurales |
| `entity-relationship.md` | `#MermaidER` | Modelos de datos y cardinalidades |
| `state.md` | `#MermaidState` | Máquinas de estado y workflows paralelos |
| `user-journey.md` | `#MermaidUserJourney` | Experiencias de usuario con métricas |
| `c4-context.md` | `#MermaidC4Context` | Vista de contexto de sistemas |
| `c4-container.md` | `#MermaidC4Container` | Arquitectura de contenedores y servicios |
| `c4-component.md` | `#MermaidC4Component` | Componentes internos y dependencias |
| `c4-deployment.md` | `#MermaidC4Deployment` | Infraestructura, despliegues y topologías |

## Uso recomendado
- Referencia estos archivos por su hashtag en el prompt de sistema para mantener consistencia con el conocimiento disponible.
- Mantén la sintaxis validada en Mermaid Live Editor antes de agregar nuevos ejemplos.
- Al añadir nuevos tipos o variantes, extiende esta tabla para que el mapeo siga siendo claro.
